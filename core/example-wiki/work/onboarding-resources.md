# Onboarding Resources

**Audience**: New hires joining the Platform Engineering team.

## Start Here

- Welcome Doc: `Google Drive > Team > Onboarding > Day 0 Overview`
- Request access checklist (Linear ticket template `OPS-112`)
- `tools.md` for day-to-day software and login instructions

## Systems Overview

- **Authentication Service** – Architecture doc (`Confluence #Auth-124`)
- **API Gateway** – Rate limiting policies, feature flags, rollback strategy
- **Monitoring** – Datadog dashboards (`shared/platform`) + alert roster

## Learning Plan

- Week 1: Pairing sessions with a buddy (see Linear `PE-Buddy`)
- Week 2: Shadow on-call; review incident retros
- Week 3: Ship first PR, present learnings at team demo

## Culture & Rituals

- Monday metrics review (9:30am, Zoom link in calendar invite)
- Wednesday engineering lunch & learn
- Friday demos + wins shout-outs
