## Weekly Structure

- **Mon**: Recovery run + mobility (30–40 min easy, drills post-run)
- **Tue**: Interval session (e.g., 6x800m @ 10K pace)
- **Wed**: Midweek medium-long run (8–10 miles) + strength

- **Fri**: Easy run or cross-train (bike or swim)
- **Sat**: Long run progression every other week
- **Sun**: Rest + foam rolling + meal prep
